# book-system-management
Full backend for book management system
## Technologies
- Java 8 above
- Spring: security, jpa/hibernate, lombok, sendmail
- Grandle, Swagger, RabbitMQ
- Docker: run to rabbitmq
- Database: PostgreSQL
